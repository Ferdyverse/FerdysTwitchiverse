Welcome to the Ferdyverse API

Your gateway to managing Twitch-integrated events and overlays.

API Overview

The Ferdyverse API allows you to manage overlays, Twitch events, and more. Below are some useful endpoints.

Available Endpoints

  • WebSocket Connection: /ws
  • Overlay Display: /overlay
  • Send Data to Overlay: /send-to-overlay
  • Fetch Overlay Data: /overlay-data
  • Manage Clickable Objects: /get-clickable-objects
  • Trigger Alerts: /alerts
  • System Status: /state

Useful curl Commands

Clickable Objects

Add Clickable Icon

curl -X POST "http://localhost:8000/send-to-overlay" \
     -H "Content-Type: application/json" \
     -d '{ "clickable": {
            "action": "add",
            "object_id": "star-icon",
            "x": 300,
            "y": 400,
            "width": 60,
            "height": 60,
            "iconClass": "fa-star"
        }
    }'

Add Clickable HTML

curl -X POST "http://localhost:8000/send-to-overlay" \
     -H "Content-Type: application/json" \
     -d '{ "clickable": {
            "action": "add",
            "object_id": "html-test",
            "x": 400,
            "y": 600,
            "width": 300,
            "height": 60,
            "html": "Ich bin ein Test!"
        }
    }'

Get All Clickable Objects

curl -X GET "http://localhost:8000/get-clickable-objects"

Remove Clickable Object

curl -X DELETE "http://localhost:8000/remove-clickable-object?object_id=star-icon"
